<h2 class="header2">Description</h2>
<p class="text-content">
A mid level react project aimed at utilizing reducer and custom api hooks to create a working quiz application. Originally a follow along project as part of the Ultimate react course I decided to expand upon it utilizing everything I had learnt to make a functioning quiz app. It works by taking an API call from the open trivia database, utilizing user inputs, passing the user data through along with our call and returning multiple choice questions we can use for the quiz, it will then, utilizing a timer, time the player to complete the quiz as quickly and as accurately as possible, giving them a score based on the question. At the end the user will be given a final score.

<h2>Installation</h2>

First clone this repo

```sh
git clone
```
NPM install to get the dependancies

```sh
npm install (or pip install)
```
Use npm start to start the local server / test the app

```sh
npm start or press run
```
